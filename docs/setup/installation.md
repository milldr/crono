# Installation

## Prerequisites

- **Node.js 18+** — [Download](https://nodejs.org/)
- **Kernel.sh account** — [Sign up](https://kernel.sh)

## Install via npm

```bash
npm install -g crono
```

## Install from source

```bash
git clone https://github.com/milldr/crono.git
cd crono
npm install
npm run build
npm link
```

## Verify Installation

```bash
crono --version
```
